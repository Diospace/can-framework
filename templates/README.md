This is a new project scaffolded with the `@decaspace/can-framework` CLI.

## 🚀 Getting Started

Follow these steps to set up and run your development server:

1.  **Navigate to your project directory:**
    ```bash
    cd my-new-app # Replace 'my-new-app' with your project name
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

3.  **Start the development server:**
    ```bash
    npm run dev
    ```
    Your application will be available at `http://localhost:3000` (or another port if specified).
